## 外部音乐（来源于其它平台）

基本的功能（以下所有都有的）

- [x] 搜索 = 【单曲 + MV + 专辑】
- [x] 听歌 = 【音频资源 + 歌词 + 专辑封面】
- [x] 看MV = 【视频资源 + 封面】
